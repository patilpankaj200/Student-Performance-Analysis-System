# Challenge 1 – Student Performance Analysis Dashboard

## Project Overview
This project analyzes student academic performance using Python data analysis tools. The main objective is to explore student score data, identify patterns, and generate useful insights through visualization and statistical analysis.

## Objectives
- Load and explore the dataset
- Clean and preprocess the data
- Perform exploratory data analysis (EDA)
- Create visualizations to understand trends
- Summarize key insights from the data

## Tools and Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Analysis Performed
The dataset is first loaded and examined to understand its structure. Data cleaning techniques are applied to handle missing values or inconsistencies. Exploratory data analysis is then performed to study score distributions and relationships between different subjects. Visualization techniques such as histograms, box plots, and heatmaps are used to represent patterns and correlations in the dataset.

## Key Insights
- Student scores vary across subjects such as math, reading, and writing.
- Certain subjects show strong correlations in performance.
- Data visualization helps identify trends and performance patterns among students.
